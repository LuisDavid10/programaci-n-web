TEXTO

Negrita <b> bold <strong>:
Se utiliza para renderizar texto en negrita
Itálica <i> italic <em> emphasized:
Se utiliza para renderizar texto en italica
Subrayado <mark>:
Se utiliza para renderizar texto con subrayado.
Resaltado <mark>:
Se utiliza para destacar texto con un fondo y borde coloreados.
Borrado <del>:
Se utiliza para renderizar texto borrado o tachado.
Pequeño <small>:
Se utiliza para renderizar texto en un tamaño menor que el normal
Superindice <sup> superscript:
Se utiliza para renderizar texto con un superindice (por ejemplo, 2)
Subíndice <sub> subscript:
Se utiliza para renderizar texto con un dubíndice (por ejemplo, H2O)
Código <code>subscript:
Se utiliza para renderizar texto como un código fuente (por ejemplo, programación).
Tecla (kbd):
Se utiliza para renderizar texto que representa una tecla o secuencia de teclas (por ejemplo, Ctrl+C)
Muestra de código <samp>:
Se utiliza para renderizar texto que representa una muestra de codigo o salida de un programa.
Variable <var>:
Se utiliza para renderizar texto que representa una variable en un contexto matemático o científico.
